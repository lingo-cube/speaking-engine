## Context

V2.1 底部栏训练模式：句子文本预览 + Listen/Record 并排 + 录音后 Play/Re-record 挤在同一 flex-1 空间。播放按钮附带进度条。NextQuestionBar 在桌面多余、移动端不适用。

## Goals / Non-Goals

**Goals:**
- 聆听区和录音区独立全宽卡片
- 按钮纯图标，无文字标签
- 录音区三态在自身卡片内完整展示
- 段落缩进
- 移除冗余元素（句子预览文本、NextQuestionBar）

**Non-Goals:**
- 不改变阅读模式
- 不改变全文播放逻辑
- 不改变录音功能逻辑

## Decisions

### 训练模式布局

```
┌──────────────────────────────────┐
│  [×]                  ← 右上关闭│
│                                  │
│  ┌────────────────────────────┐  │  ← 主色卡片（play zone）
│  │                            │  │
│  │            ▶               │  │  ← 纯图标，48px 居中
│  │                            │  │
│  │  (播放中: ▶ → ⏸)          │  │
│  │                            │  │
│  └────────────────────────────┘  │
│                                  │
│  ┌────────────────────────────┐  │  ← 红色卡片（record zone）
│  │                            │  │
│  │            🎤              │  │  ← 纯图标，48px 居中（ready）
│  │                            │  │
│  │  (录音中: 🎤 → ⏹ + 00:05) │  │
│  │  (录完: ▶ playback | ↻)   │  │
│  │                            │  │
│  └────────────────────────────┘  │
│                                  │
│  ▶ Full · 2/3                  │  ← 折叠全文链接
└──────────────────────────────────┘
```

### 录音区三态

| 状态 | 显示 |
|------|------|
| ready | 🎤 图标居中 |
| recording | ⏹ 图标 + "00:05" 计时器 |
| done | ▶ playback（大按钮）+ ↻（小按钮）并排 |

### 段落缩进

ArticleContent 每个 `<p>` 添加 `text-indent: 2em`。
