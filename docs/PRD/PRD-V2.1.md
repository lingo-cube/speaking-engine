# Speaking Engine V2.1 — PRD
## 播放交互优化 & 界面美化 & 学习流重构

---

## 1. 底部控件栏重构

### 1.1 双模式设计

**阅读模式**（未选中句子）：底部仅显示全文播放按钮 + 音频进度条
**练习模式**（选中句子）：底部展开 Listen + Record 大按钮，全文折叠为小字

### 1.2 模式切换

| 操作 | 结果 |
|------|------|
| 点击文章句子 | → 练习模式 |
| 点击底部 [×] | → 阅读模式 |
| 点击顶栏 ▶ | → 下一题（自动进入阅读模式） |
| 点击顶栏 ◀ | → 上一题（自动进入阅读模式） |

### 1.3 播放互斥

全文播放和单句播放互斥：启动一个自动停止另一个。永远只有一个音频源发声。

### 1.4 真实音频进度条

句子点阵 `●●●○` → 真实音频进度条 `████░░░░ 65% 0:08/0:15`

## 2. 句子选中视觉

半透明背景叠加 + 左边条指示器，在浅色/深色主题下均可读。

浅色: `rgba(79,70,229, 0.12)` 背景 + 3px indigo 左边条
深色: `rgba(129,140,248, 0.15)` 背景 + 3px light-indigo 左边条

## 3. 学习流重构

### 3.1 问题导航

| | 桌面端 | 移动端 |
|---|--------|--------|
| 问题列表 | 侧栏始终可见 | 顶栏 ☰ 弹出 bottom sheet |
| 下一题 | 侧栏点下一个 | 底部预览条 [→ Next] |
| Topic 切换 | 顶栏标签下拉 | 顶栏标签下拉 |
| 进度 | 侧栏高亮 | 顶栏 ● N/M |

### 3.2 学习闭环

阅读 → 选句练习 → [×] 回阅读 或 ▶ 下一题

### 3.3 Topic 退居标签

Topic 从首页降级为顶部小标签 + 下拉选择器。
